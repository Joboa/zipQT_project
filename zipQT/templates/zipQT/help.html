zipQT converts a .txt file to QTI zip file for multiple choice questions.
NOTE: Only .txt files should be uploaded.
How to use zipQT
  • On the 'Home Page', click on Go to convert file (Read "Before choosing a file")
  • Click on choose file and then upload
  • This therefere generates a QTI zip file and download automatically
Before choosing a file

The format of the .txt file should be in the form as shown below with the correct answers indicated with (*)

------------------------------------------------------------------------------
GROUP
pick: 1
points per question: 1

1. Mango is a fruit.
*a) true
b) false

2. Sum of 3 and 5 is
a) 2
*b) 8

END_GROUP
